# Homepage Images Guide

## Where to Add Images:

### 1. Hero Section Background
**File Location**: `assets/images/hero-bg.jpg`
**Usage**: Main banner background
**Size**: 1920x1080px recommended
**Code**: Will be added to `.hero-section` CSS

### 2. Hero Side Image
**File Location**: `assets/images/fitness-model.png`
**Usage**: Person/product image next to hero text
**Size**: 500x600px recommended
**Code**: Will be added to hero HTML structure

### 3. Category Icons
**File Location**: `assets/images/categories/`
- `whey-protein.png`
- `mass-gainer.png`
- `pre-workout.png`
- `vitamins.png`
- `amino-acids.png`
- `fat-burners.png`
**Size**: 64x64px or 128x128px
**Code**: Will replace emoji icons

### 4. Feature Images
**File Location**: `assets/images/features/`
- `premium-quality.png`
- `fast-delivery.png`
- `expert-support.png`
**Size**: 100x100px recommended
**Code**: Will be added to features section

### 5. Brand Logos
**File Location**: `assets/images/brands/`
- `brand1.png`
- `brand2.png`
- `brand3.png`
**Size**: 150x80px recommended
**Code**: New brands section will be created

## Instructions:
1. Upload your images to the specified folders
2. Tell me which type of images you want to add
3. I'll update the code accordingly
4. Images will be mobile responsive automatically

## Supported Formats:
- JPG/JPEG (for photos)
- PNG (for logos/icons with transparency)
- WebP (for better compression)
- SVG (for scalable icons)
